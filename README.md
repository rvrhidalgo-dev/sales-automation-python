# 📊 Sales Report Automation (Python)
Desarrollado por **Irving Hidalgo** (@rvrhidalgo-data)

## 🎯 Objetivo
Este proyecto automatiza la transformación de datos de ventas crudos en un reporte de Excel profesional con KPIs, limpieza de datos y visualización automática.

## 🛠️ Tecnologías
- **Python / Pandas**: Limpieza y análisis de datos.
- **Matplotlib**: Generación de gráficos dinámicos.
- **XlsxWriter**: Creación de reportes de Excel con formato empresarial.

## 📈 Resultados del Script
- Limpieza automática de fechas y formatos de moneda.
- Identificación de categorías estrella y ventas totales.
- Generación de un archivo `.xlsx` estructurado con tablas y estilos.

## ⚙️ Cómo ejecutar
1. Instala las dependencias: `pip install -r requirements.txt`
2. Ejecuta: `python main.py`
